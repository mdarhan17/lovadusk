# LOVA DUSK Project

LOVA DUSK is a PHP + MySQL luxury fashion ecommerce project with drop-based product launches.

## Main Features

- User registration and login
- Admin panel
- Drops and products
- Product sizes and stock
- Cart and checkout
- Razorpay payment integration
- Orders and invoices
- Coupons
- Wishlist
- Search
- Reviews and ratings
- Contact messages
- Newsletter
- Admin analytics and reports
- Activity logs
- Database backup
